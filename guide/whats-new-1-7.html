    What's new in version 1.7

    Version 1.7 of the i2 Notebook API enhances the customization options for the application ribbon, allowing the removal of existing ribbon tabs. This allows the application ribbon UI to be fully customized, replacing the standard ribbon tabs with your own arrangement of tools and commands to fit your specific requirements.

    The new version also adds a flag to the security dimension values of records to show when a value has been "suspended" due its removal from the current security schema.
