    schema.IChartPropertyType.possibleValues property

    Gets the possible values for a property that has the property type.

    Signature:

    readonly possibleValues?: data.IReadOnlyCollection<IPossibleValue>;

    Returns: data.IReadOnlyCollection<IPossibleValue>

    Introduced: version 1.0

    Remarks

    This is provided only for property types whose logical type is suggestedFromList or selectedFromList. For selectedFromList properties, the value must be one of the possible values; for suggestedFromList properties, the possible values are just suggestions that can be provided to users.
